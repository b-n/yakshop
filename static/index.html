Yakshop

Welcome to the yakshop!

Endpoints

`GET /yakshop/herd/T` where `T` is the day number you wish to view

`GET /yakshop/stock/T` where `T` is the day number you wish to view

`POST /yakshop/stock/T` where `T` is the day number you wish to view.
A request body should be JSON (Content-Type: application/json) with the following structure:
{ "customer" : "Medvedev", "order" : { "milk" : 1100, "skins" : 3 } }
